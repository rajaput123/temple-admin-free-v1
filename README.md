# NM-Temple Admin Platform

A comprehensive temple management system built with modern web technologies.

## Project Info

**Repository**: https://repo.corp.infokalash.com/keehoo-products/namaha-unified/nm-temple-admin-platform.git

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Step 1: Clone the repository
git clone https://repo.corp.infokalash.com/keehoo-products/namaha-unified/nm-temple-admin-platform.git

# Step 2: Navigate to the project directory
cd nm-temple-admin-platform

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```

## Features

- **Temple Structure Management**: Manage temples, child temples, sacred entities, zones, halls, rooms, and counters
- **PR & Communication**: Announcements, devotee messaging, media management, crisis communication, social media
- **People & HR**: Employee management, organization structure, shifts, leave, attendance, expenses
- **Rituals & Darshan**: Offerings, daily schedules, festivals, priest assignments
- **Stock & Inventory**: Items master, stock entries, purchase orders, godowns
- **Kitchen & Prasad**: Recipe management, production planning, prasad distribution
- **Seva Counter Booking**: Real-time booking system with cash management
- **Asset Management**: Asset tracking with CV verification, custody, maintenance
- **Projects Management**: Construction, renovation, restoration project tracking
- **Platform Management**: Temple onboarding, configuration, subscriptions
